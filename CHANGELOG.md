# Changelog

All notable changes to Skyline will be documented in this file.

## [0.1.0] - 2026-03-23

### Added
- Fleet Overview Panel with MapLibre world map, radar display, aircraft registry table
- Maintenance Intelligence Panel with overdue alerts, 90-day timeline, fleet pattern detection
- Procurement & Inventory Panel with two-track pipeline (planned + AOG bypass), low stock alerts
- Cost Analysis Panel with YTD gauges, monthly trends, cost breakdown, AOG burn tracker
- Tower AI copilot with 9 Claude API tools (fleet, maintenance, procurement, inventory, MEL, compliance, reports, anomalies, dashboard control)
- MEL Tracker with category-based deferral countdown timers
- AD/SB Compliance Board for regulatory tracking
- Dispatch Reliability metric with trend charts
- AOG Timeline with revenue impact calculator
- Multi-Station Inventory view with cross-station transfer recommendations
- HUD design system with aviation blue/amber palette
- PIN-gated authentication with cinematic boot sequence
- Guided tour for first-time users
- Mock data: 30 aircraft, 70 maintenance records, 25 MEL items, 50 procurement orders, 100 inventory items, 164 scheduled flights, 12 months of cost data
